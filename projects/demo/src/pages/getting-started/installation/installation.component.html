Local development

Prerequisites

To get started locally you should have a few things:

  • Node.js installed.
  • Angular installed.

A working knowledge of Angular is helpful. You don't need to be an expert, but you should be comfortable with the basics.

1. A new Project

First, spin up a new Angular project. If you would like to use Taiga themes , less stylesheets are recommended.

2. Add NgDrawFlow

NgDrawFlow is published on npm as ng-draw-flow, so go ahead and add it next.

3. Run it!

Lastly, spin up the dev server and we're good to go!
