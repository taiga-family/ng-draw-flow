This guide explains the core concepts of NgDrawFlow. Three things you’ll be using a lot in NgDrawFlow are nodes, connections, and connectors.

Nodes

Concept

A node is used to display data. A node in NgDrawFlow is an Angular component. This means it can display anything you desire. Each node has X and Y coordinates that refer to the point at the center of the node, relative to the workspace. To create a node, you need to extend your Angular component from our DrawFlowBaseNode component.

Actions

Users can interact with your node. Currently, we support

  • Deleting - To delete a node, select it and press Delete or Backspace
  • Dragging - To drag a node, click and hold the node, then move it to the desired location.

Customization

You can customize your nodes as you wish.Everything what is possible in an Angular component, is possible in our node. Some things you can do are:

  • Render form elements
  • Visualize data
  • Support multiple connectors
  • Add custom styles & design different nodes for different purposes in your flow

Requirements

Nodes require a unique name and id (for example node-1-input-1 ).

More information can be found in the nodes documentation .

Connectors

Concept

A Connector is used to link nodes together. Connectors are divided into two types: input connectors and output connectors.

  • input connectors – Receive connections from the output connectors of other nodes.
  • output connectors – Serve as starting points for connections to the input connectors of other nodes.

Actions

Users can interact with connectors. Currently, we support

  • Connecting via Dragging - To connect an input with an output connector, click and hold the output connector, then move it to the desired input connector.
  • Deleting - To delete a connector, select it and press Delete or Backspace .

Customization

By default connectors are displayed as blue circles but the appearancean be changed using css variables. Some styling you may use are

  • Change the color
  • Change the size of the connector
  • Change the shape of the connector

When creating a custom node, you may use as many connector handles as you need.

Requirements

Connectors require a unique name starting with its nodeId (for example node-1-input-1 ). More information can be found in the connector documentation .

Connections

Concept

A Connection links two nodes together. It is represented by a Bezier curve (a fancy parametric curve). Technically, the connection is an SVG path, allowing its line thickness and color to be adjusted through CSS variables.

Actions

Users can interact with connections. Currently, we support

  • Deleting - To delete a connection, select it and press Delete or Backspace .

Requirements

Connections require an input connector and an output connector .
