Overview

Connection lines define how nodes are visually linked in NgDrawFlow. The library provides two styles: Bezier and SmoothStep . Select the style via the connection.type option.

SmoothStep

SmoothStep edges use horizontal and vertical segments joined by rounded bends, making them suitable for orthogonal layouts.

  • curvature : radius of each bend in pixels.
